Добро пожаловать на портал описания FraudBusters!

FraudBusters - система для анализа и фильтрации мошеннических операций.

Для кого и зачем создан данный сервис?

Система подойдет вам если необходимо:

  • уменьшить количество чарджебоков по платежам
  • уменьшить фон фродовых операций
  • анализировать поток ваших транзакций

С чего начать?

Шаг 1 Обратитесь к синтаксису языка обзор.

Шаг 2 Просмотрите список доступных:

  • функций;
  • возможных результатов применения шаблона;
  • полей для группировки;
  • доступных единиц времени

Шаг 3 Обратитесь к примерам для лучшего понимания стуктуры языка и понимания синтаксиса.
